Linux命令汇编

你可以用 man 命令查询文档(如man man)

  • 转换用户 su sudo

  • 重启关机 reboot poweroff

  • 输出 echo printf

  • 文件内容 vim cat more head tail tr grep cut diff uniq sort

  • 文件目录管理 wegt wd cd ls tree touch mkdir cp rm dd file tar

  • 文件查找 find locate whereis which

  • 任务管理器 btop(需下载) top

  • 时间 date timedatectl

  • 进程 ps pstree pidof nice kill killall

  • 系统负载 uptime free

  • 网络环境 ifconfig ping tracepath netstat
